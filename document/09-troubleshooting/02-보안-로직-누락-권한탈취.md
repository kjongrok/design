# Trouble Log

## 1. 문제 요약

- 제목: 보안 로직 누락에 따른 권한 탈취/잠금 버그
- 프로젝트: BidMatch
- 영역: 보안/인증 (백엔드)
- 심각도: Critical
- 상태: 해결됨
- 발견 시점: 2026-06-14
- 담당자: 김종록

## 2. 발생 상황

1. 관리자가 특정 악성 유저의 상태를 '정지(blocked)'로 변경했음에도 불구하고, 해당 유저가 카카오/구글 소셜 로그인을 통해 버젓이 서비스에 접속할 수 있는 문제 발견.
2. 최고 관리자가 실수로 본인의 권한을 일반 유저(USER)로 내리거나 본인을 정지시켜 영영 관리자 페이지에 접속할 수 없게 되는 치명적인 조작 실수 가능성 발견.

## 3. 에러 메시지 또는 증상

- 정지된 사용자가 소셜 로그인 시도 시 정상적으로 로그인되어 서비스 이용이 가능함.
- 관리자가 본인 계정의 상태를 변경하여 스스로 접근 권한을 잃는 상황 발생 가능.
