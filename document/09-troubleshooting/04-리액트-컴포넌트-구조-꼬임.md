# Trouble Log

## 1. 문제 요약

- 제목: React 컴포넌트 구조 꼬임으로 인한 빌드(Build) 실패
- 프로젝트: BidMatch
- 영역: 인프라/프론트엔드
- 심각도: High
- 상태: 해결됨
- 발견 시점: 2026-06-15
- 담당자: 안민영

## 2. 발생 상황

관리자 화면에 '회원 관리' 탭 기능을 덧붙인 후 클라우드타입에 배포를 시도했으나, 로컬 개발 서버에서는 보이던 화면이 배포 단계에서 `npm run build` 스크립트 에러(Syntax Error)를 뿜으며 중단됨.

## 3. 에러 메시지 또는 증상

배포 환경 빌드 과정에서 Syntax Error 발생 및 빌드 실패.

## 4. 재현 방법

- 배포 스크립트(`npm run build`) 실행

## 5. 원인 분석

JSX 코드에서 복잡한 조건부 렌더링(삼항 연산자)을 수정하던 중 닫는 태그(`</div>` 및 `</>`)가 일부 누락되거나 잘못된 위치에 배치되어 컴포넌트 트리가 완전히 붕괴된 상태였음.

| 가설 | 확인 방법 | 결과 |
| --- | --- | --- |
| - | 빌드 에러 로그(Line number) 추적 | Layout 닫는 태그 누락/오류 확인 |

## 6. 해결 방법

JSX 구조(태그) 원상 복구 및 수정.

- 수정 파일: 관리자 화면 관련 컴포넌트 JSX 파일 추정
- 수정 내용: 손상된 Layout 닫는 태그 복구 
- 관련 커밋: -
- 관련 PR: -

빌드가 에러 없이 통과(`npm run build` 성공)되며 배포 정상화 완료.
