# BidMatch 프로젝트 진행 상태 (PROGRESS.md)

## 📌 프로젝트 개요
**BidMatch**는 매일 등록되는 나라장터(G2B) 입찰 공고를 자동으로 수집하여, 사용자가 설정한 맞춤 조건(키워드, 지역, 업종 등)에 매칭하고 이를 이메일 다이제스트 알림으로 전송해 주는 B2B SaaS 서비스입니다.

---

## 🛠 완료된 작업 내역 (Completed)
### 1. 백엔드 (Flask & MariaDB)
- **Flask API 서버 아키텍처 구축**: 기본적인 API 라우터, DB 세션 연동 및 설정 구성
- **DB 스키마 자동 동기화 (`schema_manager.py`)**: MariaDB 상태를 감지하여 유효한 스키마를 보강하는 모듈 설계
- **나라장터 공고 수집 배치 작업 (`collect_bid_notices_job.py`)**: 나라장터 Open API 연동, 용역 및 물품 공고 수집(1시간 단위 스케줄러 포함)
- **백그라운드 워커 구동**: `APScheduler` 기반으로 백그라운드 수집 실행 기능 지원

### 2. 프론트엔드 (React & Vite 기반 단독 시연용 디자인 목업 구축)
- **인증 및 내 정보 설정**: 이메일 기반 회원가입/로그인, 마이페이지 연동
- **공고 정보 3대 메뉴화**: 입찰공고 목록, 사전규격 목록, 개찰결과·낙찰정보 화면 구축
- **입찰제안서 지원**: 특정 공고 선택 후 RFP 기반 제출서류 체크리스트, 제안전략 FAQ, 템플릿 다운로드 연동
- **관심 공고함**: 관심공고 스크랩 기능 구현, 관심공고함([InterestNotices.jsx](file:///c:/BidMatch/front-end/src/pages/InterestNotices.jsx)) 목록 및 상세 해제 인터랙션 동기화
- **공고 상세 탭화 개편**: 기본 정보, 자격 자가진단(면허/지역/실적 대조), AI 매칭 분석, 유사 공고 비교 탭 탑재
- **자사 실적 관리**: 마이페이지에 실적 관리 탭을 추가하여, 등록한 계약 금액이 공고 상세의 자격 진단 결과에 즉시 실시간 동기화되도록 연동
- **어드민 자격진단 검수 큐**: AI 조건 추출 정확도를 보강하기 위해 어드민 대시보드 내 검수 큐 탑재
- **Vite 빌드 통과**: 모든 문법 및 인터랙티브 동작 검증 완료 (빌드 정상 완료)
